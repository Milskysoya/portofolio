# Pixel Cybersecurity Portfolio

A personal portfolio website with a **Green Terminal / Retro Pixel** aesthetic, designed for a Cybersecurity & Electrical Engineering student.

## Features

- **Green Terminal Theme**: Pure CSS green-on-black color scheme with scanline effects.
- **Interactive Console**: A functional JavaScript-based terminal where users can type commands like `whoami`, `ls skills`, and `cat about_me.txt`.
- **Responsive Design**: Adapts for mobile and desktop screens.
- **Pure Technology**: Built with 100% Vanilla HTML, CSS, and JavaScript. No external frameworks.

## Quick Start

1.  Clone or download this repository.
2.  Open `index.html` in any modern web browser.
3.  Type `help` in the interactive terminal section to see what you can do!

## Structure

-   `index.html`: Main structure and content.
-   `style.css`: All visual styling, animations, and responsive rules.
-   `script.js`: Logic for the typing effect, section revealing, and interactive terminal commands.

## Customization

-   **Profile Picture**: Place a file named `profile.jpg` in the project folder. The code uses `profile.jpg` by default.
-   **Terminal Commands**: Edit the `processCommand` function in `script.js` to add more secrets or commands.

## Deployment to Vercel

1.  Push this code to a GitHub repository.
2.  Go to [Vercel.com](https://vercel.com) and log in.
3.  Click **"Add New"** > **"Project"**.
4.  Import your GitHub repository.
5.  Click **"Deploy"**. No build settings or environment variables are needed for this static site.
